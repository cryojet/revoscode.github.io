# Unsupported Mods

PojavLauncher does not support every mod available to the Minecraft world. A non-exhaustive list is provided below:

  • Sodium (opens new window) (not supported by GL4ES, works with ANGLE and virglrenderer)
  • Rubidium (opens new window) (same as Sodium ^)
  • Chlorine (opens new window)
  • Create (opens new window)
  • Scannable (opens new window) (Works on 1.12)
  • Colormatic (opens new window)
  • ItemPhysic (Lite (opens new window) works)
  • RandomPatches (Forge) (opens new window) (Fabric) (opens new window)
  • Immersive Vehicles (opens new window) (Forge Only)
  • MrCrayfish's Gun Mod (1.16.5) (opens new window)
  • ReplayMod (opens new window) (FFMPEG wrongly placed)
  • Dynamic Surroundings (opens new window) (White textures on 1.12)
  • Sex mod (thanks @Adiixyz)
  • Essential (opens new window)
  • Skytils (opens new window) (1.3.0-pre5 works)
  • Patcher (opens new window) (1.6.1 works)
  • Immersive Engineering (opens new window) (Works on 1.12)

# Unsupported Clients

  • BatMod (opens new window)
  • LabyMod (opens new window) (only 1.8.9)
  • MysteryMod (opens new window) (Not fully tested, but works as it's a Labymod "fork")
  • Lunar Client (opens new window) (Obvious reason ".exe")
  • Badlion Client (opens new window) (same as Lunar)
